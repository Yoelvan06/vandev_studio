Game engines that i use to make games
Max2D This app is completely free and good enough to make simple mobile games. I use this to make a lot of mobile games, Redball adventure is one of the example. However, this app is only available for mobile devices. But hey, its free mobile game engine
Adventure game studio One of my favourite game engine ever. If you wanna make your own point & click adventure game, this game engine fit perfectly for you. it's free and also there are some available templates for beginners to learn.
RenPy Visual novel is a real thing nowadays, and RenPy is one of the most recommended for making visual novel games. it has a lot of features that you need for visual novel. Heck, you can also create a minigame inside your visual novel. And the good thing is this app is free
RPG maker There are some different version with this engine, you can choose anything you want. They're basically have almost the same features as each other, just slightly different from the assets side. This software will 100% help you making your own rpg game, "Our Princess" is one of the example that i made by using rpgmv. However its not free at all, but you can try the trial for free.
